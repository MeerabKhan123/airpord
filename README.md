# Airpord

## Overview
**Airpord** is a beginner-friendly front-end project showcasing my skills in **web development**.  
This project was created as part of my **1.5-year MERN Stack course** and a **3-month internship**.  

It demonstrates my understanding of **HTML, CSS, and JavaScript**, as well as my ability to design **responsive and interactive web pages**.
